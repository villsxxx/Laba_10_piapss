# Лабораторная работа №10: Моделирование информационной системы университета

Данный проект представляет собой объектно-ориентированную модель информационной системы университета, реализованную на языке **Java**. Система отражает структуру взаимодействия между сотрудниками, факультетами, институтами, курсами и научными проектами.

---

##  Основные компоненты

- **`University`** — корневой объект, объединяющий факультеты.
- **`Faculty`** — содержит декана и институты.
- **`Institute`** — включает научных сотрудников и связанные проекты.
- **`Employee`** — абстрактный базовый класс для всех типов сотрудников:
  - `Dean` (декан)
  - `Lecturer` (преподаватель)
  - `ResearchAssociate` (научный сотрудник)
- **`Participation`** — ассоциативный класс, связывающий научного сотрудника с проектом и отслеживающий отработанные часы.

---

##  Архитектура системы

Проект демонстрирует ключевые принципы ООП:

- **Наследование**: класс `Employee` служит базой для всех ролей сотрудников.
- **Композиция**: университет → факультеты → институты → сотрудники.
- **Ассоциация**: связь между `ResearchAssociate` и `Project` через класс `Participation`.
